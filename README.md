# Wiki
  A wikipedia-style website done as a part of the CS50 Harvard course.
## Technologies
- Django
- sqlite
- markdown2 python package
## Installation
- Clone the reposetory to your local machine
- In the local path run the command
  ```
  pip install -r requirements.txt
  ```
- To run the project run the command
  ```
  python manage.py runserver
  ```
- Go to the specified url shown in the terminal
